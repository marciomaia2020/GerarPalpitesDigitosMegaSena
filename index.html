🎯 Estratégia Minimalista - Regras Detalhadas

📊 Como Funciona o Método:

1. Extração de Dados: Sistema pega informações oficiais do próximo sorteio

2. Aplicação de Regras: Aplica regras matemáticas simples nos números extraídos

3. Geração de Dezenas: Cria dezenas válidas (01-60) baseadas nos cálculos

4. Validação: Remove duplicatas e ajusta para a quantidade solicitada

🔧 Detalhamento das 7 Regras:

R1 - Direto

Conceito: Usar números que aparecem "puros" nos dados

Exemplo: Data 21/08 → dezenas 21, 08

R2 - Soma

Conceito: Somar dois números da base

Exemplo: 21 + 25 = 46

R3 - Dobro/Multiplicação

Conceito: Dobrar ou multiplicar números

Exemplo: 21 × 2 = 42

R4 - Diferença

Conceito: Subtrair valores entre si

Exemplo: 35 - 21 = 14

R5 - Inversão

Conceito: Espelhar números

Exemplo: 25 → 52

R6 - Vizinhos

Conceito: Números próximos

Exemplo: 35 → 34, 36

R7 - Soma Cruzada

Conceito: Somar elementos da mesma categoria

Exemplo: Dia + Mês = 29
